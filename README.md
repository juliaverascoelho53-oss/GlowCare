##Sobre o projeto

Este projeto foi desenvolvido com o objetivo de aprimorar meus conhecimentos em **HTML, CSS e JavaScript**. Durante o desenvolvimento, realizei diversas pesquisas para aplicar corretamente conceitos de estruturação, estilização e interatividade em páginas web.
O site possui caráter informativo, com foco em **cuidados com a pele**, oferecendo dicas úteis para diferentes tipos de pele. Além disso, conta com uma interação simples com o usuário por meio de perguntas, com o objetivo de sugerir as melhores opções para o **melhoramento da pele** de forma personalizada.
________________________________________
##Funcionalidades
* Exibição de dicas sobre cuidados com a pele
* Interface simples e intuitiva
* Interação com o usuário através de perguntas
* Sugestões personalizadas com base nas respostas
